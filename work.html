Grading Software

Axolotl Development designed a tool to facilitate grading labs in introductory computer science courses. This includes an interface that allows for viewing, commenting and scoring code better than the current system. It also includes a solution for backing up, sharing, and exporting work.

Built in C++, the grading software features easy installation with compatibility on multiple machines, a dual view for users to provide feedback on submissions while grading, in-program rubric creation, easy file uploading, result exportation, and a fully implemented data archival mechanism.

CPU Simulator

I built a simulator for a simple CPU and its associated main memory. The simulator took in a file of register transfer notation to specify the implementation of the instruction set architecture as well as a configuration file to define word-size, bus size, ALU functionality, and number of general registers. I also developed a graphical user interface to reflect the progression of the simulation. The main memory was byte addressable, bounded by a specific word size. The data structures used to develop the simulation, as well as formatting such as the structure of the instructions, were designed to enable accuracy as well as clarity for the user of the simulation. I also ensured that the clock did not use boolean logic and that the fetch-execute cycle demonstrated abstraction. Object-oriented design was leveraged in the development of the simulator.
